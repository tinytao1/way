CSS position: relative

In CSS, position: relative moves an element "relative" to its original position in the document flow, without affecting the space it originally occupied.

1. It keeps its original "Gap"

Unlike absolute, which removes an item from the flow, relative keeps the original space reserved. Use the slider to offset the blue box.

2. It acts as an "Anchor"

This is the most common use. Setting a parent to relative ensures its absolute children stay contained inside it.

Key Takeaways

  • ✓ Visual Offset: You can use top, right, bottom, and left to nudge the element.
  • ✓ Space Preservation: It doesn't collapse the layout. Other elements act as if the relative item never moved.
  • ✓ Z-Index: Relative elements can use z-index to stack on top of other static elements.
  • ✓ Reference Point: It defines the "coordinate system" for any nested elements using position: absolute.
